プライバシーポリシー

最終更新日: 2026年3月12日

Choreia（以下「当プロジェクト」）が提供するWebアプリケーション（Expense、Books、Forms 等、以下「本サービス」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。

1. 収集する情報

本サービスは独自のサーバーを持たず、ユーザーのデータを収集・保存しません。

  • Googleアカウント情報（名前・メールアドレス）は、ログイン時にGoogle Identity Servicesを通じて取得し、ブラウザ上での表示にのみ使用します。
  • OAuthアクセストークンはブラウザのメモリ上にのみ保持され、外部に送信されません。

2. データの保存先

本サービスで作成されるデータはすべてユーザー自身のGoogleアカウントに保存されます。

  • 経費データ・会計データ → ユーザーのGoogle Sheets
  • レシート画像・証憑 → ユーザーのGoogle Drive
  • フォーム回答 → ユーザーのGoogle Sheets

当プロジェクトの開発者がこれらのデータにアクセスすることはありません。

3. Google APIの利用

本サービスは以下のGoogle APIを利用します。

  • Google Sheets API — データの読み書き
  • Google Drive API — ファイルの保存
  • Gemini API — レシート画像のAI読み取り

これらのAPIはユーザーが明示的に許可したOAuthスコープの範囲内でのみ動作します。本サービスのGoogle APIの利用は、Google API Services User Data Policy（Limited Use requirements を含む）に準拠しています。

4. Cookieとローカルストレージ

本サービスは設定データやテンプレートの保存にブラウザのlocalStorageを使用する場合があります。これはユーザーの端末にのみ保存され、外部に送信されません。

5. 第三者への提供

本サービスはユーザーの情報を第三者に提供、販売、共有することはありません。

6. セキュリティ

すべての通信はHTTPSで暗号化されます。OAuthトークンはブラウザセッション中のみメモリに保持され、ストレージに永続化されません。

7. お問い合わせ

プライバシーに関するお問い合わせは、GitHubの各リポジトリのIssueからご連絡ください。

8. 変更

本ポリシーは予告なく変更される場合があります。変更後の内容は本ページに掲載した時点で有効となります。
